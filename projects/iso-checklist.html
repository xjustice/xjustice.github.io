Case Study: Retro UI & Compliance Management

Posted Oct 2024 • Tech Stack: CSS3, HTML5, JavaScript, Retro UI Design

Introduction

Compliance tools are traditionally seen as dry, corporate, and visually sterile. ISO Checklist XP is an experiment in breaking that mold. I combined the rigorous requirements of ISO 17020, the international standard for the operation of various types of bodies performing inspection, with the iconic aesthetic of Windows XP.

This case study explains how I developed a professional-grade audit tool that is as nostalgic as it is functional.

Design Concept: The Windows XP Aesthetic

Retro-tech is a powerful emotional hook. By utilizing the classic Luna theme, I aimed to create an interface that feels familiar, lowering the cognitive barrier for new users navigating complex auditing tasks.

Achieving this required a deep dive into pure CSS3 for:

  • Bevels and Gradients: Mimicking the classic window headers and Taskbar buttons without using heavy image assets.
  • Iconography: Pixel-perfect recreation of early 2000s desktop icons, optimized for SVG rendering.
  • Window Management: Simulating a desktop-like experience within the browser, utilizing a z-index stack that allows for draggable and focusable checklist windows.

The ISO 17020 Framework

1. Systematic Auditing

The application is pre-populated with 140+ specific check items derived from ISO 17020. I engineered a nested JSON data structure that categorizes these into *Management Requirements, Structural Requirements, and Resource Requirements*. This ensures auditors can follow a logical, section-by-section workflow.

2. Evidence Persistence

Each checklist item supports status tracking (Pass/Fail/NA) and text-based evidence logging. To ensure reliability for professionals, I implemented IndexedDB-based persistence, meaning that user progress is saved locally after every interaction, even if the browser crashes or the tab is closed.

Technical Challenges

One of the hardest parts of this project was balancing the Nostalgic UI with Modern Usability. I had to ensure the app remained responsive on mobile while the "Taskbar" and "Start Menu" remained functional. I solved this by implementing a reactive layout engine that collapses sidebars into a mobile-friendly drawer while maintaining the XP color palette and button styles.

Conclusion

ISO Checklist XP proves that professional utility does not have to be boring. It shows how the intersection of design history and modern auditing needs can create a product that stands out in a crowded market of compliance software.
